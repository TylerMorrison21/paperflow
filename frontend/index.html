Install PaperFlow MCP (Windows, macOS, Linux)

2-minute setup

Deploy once, then use PaperFlow directly inside Claude Desktop.

~80% fewer tokens · LaTeX preserved · tables intact · citations linked

Step 1: Install 

npm install -g paperflow-mcp

Step 2: Add to Claude Desktop config (macOS / Linux) 

{
  "mcpServers": {
    "paperflow": {
      "command": "npx",
      "args": ["-y", "paperflow-mcp"]
    }
  }
}

Step 3: Add to Claude Desktop config (Windows) 

{
  "mcpServers": {
    "paperflow": {
      "command": "cmd",
      "args": ["/c", "npx", "-y", "paperflow-mcp"]
    }
  }
}

Step 4: Restart Claude Desktop, then ask it to analyze any PDF with PaperFlow MCP.
